# 🌸 Blossom Corner – CRUD Operation Website

Blossom Corner is a responsive flower bouquet shopping website developed using **HTML, CSS, and JavaScript**. The project demonstrates complete **CRUD (Create, Read, Update, Delete)** operations using **Local Storage** to manage flower products without a backend database.

---

## 📌 Features

### 👨‍💼 Admin Module
- Add new flower products (Create)
- View all products (Read)
- Update product details (Update)
- Delete products (Delete)
- Product data stored in Local Storage

### 👤 Customer Module
- User Registration & Login
- View product details
- Add products to Cart
- Place Orders
- Order History
- Contact and About Page
- Delivery Information

## 🔄 CRUD Operations by Admin

### ✅ Create
- Admin can add new flower products.
- Product information is stored in Local Storage.

### 📖 Read
- All stored flower products are displayed on the Products page.

### ✏️ Update
- Admin can edit existing product details.

### ❌ Delete
- Admin can remove products from the product list.
